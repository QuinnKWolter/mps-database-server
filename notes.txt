Python Environment
==================

Install Anaconda 64-bit from:

  https://store.continuum.io/cshop/anaconda/

Upgrade Conda and Ipython packages


  $ conda install IPython

Install requirements for MPS database

  $ pip install -r requirements.txt


PostgreSQL Database Notes
=========================

First, create a DB named 'mps':

  $ createdb mps


Then, setup the database:

  $ ./manage.py syncdb

To list tables in the database:

  $ psql mps
  mps=# \dt


To drop database:

  $ dropdb mps



Starting a New Application
==========================

Start as usual:

  $ ./manage.py startapp cellsamples

Then, implement your models.

The First Migration
-------------------

Do the initial migration:

  $ ./manage.py schemamigration cellsamples --initial

Apply the migration to the database:

  $ ./manage.py migrate cellsamples

Changing the Model
------------------

Make new migration:

  $ ./manage.py schemamigration cellsamples --auto

Apply the migration:

  $ ./manage.py migrate cellsamples


Launching the Local MPS DB
------------------

Browse to: 

     'C:\Users\{userid}\Documents\mps'

Open command window: CMD

     $ python manage.py runserver

In Internet browser go to:

     http://127.0.0.1:8000

Dumping and Loading Data
------------------------
Save all the data in the database with:

     python manage.py dumpdata > mpsdata.json
	 
Load data into database:
	 
	 python manage.py loaddata [--ignorenonexistent] mpsdata.json
	 
	 The optional parameter '--ignorenonexistent' will ignore data for tables or fields that have been dropped.
