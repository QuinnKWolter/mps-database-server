Making a Plate Setup

  • A Plate Setup describes the protocols taken to prepare a plate
  • Selecting an Assay Layout will show a preview of the plate below
    Please note that you cannot edit the layout on this page, you will have to edit the layout itself
  • If the layout you need does not exist, click the provided link to add a new one
  • The Plate ID/Barcode can not be the same as another plate in the study

Adding Cells

  • Each row under the header Model Cells describes cells added to the plate
  • If this plate does not have any cells, do not modify this section
  • To get another row, click the Add Cells button
  • Click a magnifying glass to see a list of available cell samples then click the desired row to select that sample
  • Use 'e' for scientific notation: e.g. 1e3 = 1000.
  • If you need a cell sample that is not listed, click the provided link to add it

Flagging for Review

  • Clicking flags the current setup for review
  • Clicking Notes brings up an input dialog where you can explain why this setup was flagged
  • Clicking removes the flag from the current setup
