  • Each sheet of the excel file must follow either a Chip or Plate format

  • The format for Chip uploads is as follows:

    Chip ID [Chip ID]
    Time Time Unit Assay Object Value Value Unit QC Status
    [Time] [Time Unit] [Assay] [Object] [Value] [Value Unit] [Leave Blank to Pass]
    [Time] [Time Unit] [Assay] [Object] [Value] [Value Unit] [Any Value in this Column Marks the Reading Invalid]

  • For Plate uploads you may choose between Block and Tabular formats

  • A theoretical 3x2 6-well plate in Block format would be as follows:

    Plate ID [Plate ID] Upload Type Block
    Assay [Assay 1] Feature [Feature 1] Value Unit [Value Unit 1] Time* [Time 1]* Time Units* [Time Unit 1]*
    [Val 1] [Val 2] [Val 3]
    [Val 4] [Val 5] [Val 6]
    Assay [Assay 2] Feature [Feature 2] Value Unit [Value Unit 2] Time* [Time 2]* Time Units* [Time Unit 2]*
    [Val 7] [Val 8] [Val 9]
    [Val 10] [Val 11] [Val 12]
    *Optional (if not using time, totally exclude cells)

  • The above example in Tabular format would be as follows:

    Plate ID [Plate ID] Upload Type Tabular
    Well ID Assay Feature Value Unit Time* Time Units* Value
    A1 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 1]
    A2 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 2]
    A3 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 3]
